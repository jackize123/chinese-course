# 國語文課程地圖 · 小一到高三

單一 HTML 檔的國語文課程網站，無外部相依，可直接用瀏覽器開啟，也可掛在 GitHub Pages。

## 內容

- **十二年課程地圖**：國小低／中／高年級、國中、高中共 12 個年級，難度逐年遞進。
- **年級課程**：每年級含年度目標、核心能力、8 個教學單元（教學重點、選文、寫作訓練）、語文重點、評量方式，共 96 個單元。
- **教師版／學生版切換**：
  - 教師版顯示 100 分鐘教學流程建議、三層提問設計（事實／推論／評鑑）、作業與評量。
  - 學生版顯示每單元的學習任務與進度勾選。
- **特色課程（8 堂）**：每堂為「四小時單堂完課」，含完整時間流程（含休息）、教材清單、學生產出、講評方式與一句口訣。以作文寫作為主軸，另含閱讀策略、文言文速讀、口語表達、經典導讀。可列印教案。
- **寫作教室**：審題四問、素材銀行、論證三箭、五感描寫、以小見大、起承轉合變體、開頭六法、結尾五法。
- **語文常識庫**：修辭格 15 種、文言虛字 16 個、六書、標點符號 10 種。
- **隨堂測驗**：90 題單選題（國小／國中／高中各 30 題），依難度與題型抽題，答完即看解析。
- **考卷產生器**：依難度隨機出卷，含作文題與解答卷，可列印或存成 PDF。
- **進度備份**：學習進度存於瀏覽器 localStorage，可匯出／匯入。

## 修改方式

直接編輯 `index.html`（資料以 JavaScript 陣列內嵌在檔案底部的 `<script>` 中），commit 後 push，GitHub Pages 幾分鐘後自動更新。

## 說明

課程大綱依 108 課綱國語文領域精神編寫，選文為台灣課本常見篇目；實際授課請依所採用的教科書版本調整。
